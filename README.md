# LynxOS Consulting

## Descripción

Proyecto FullStack desarrollado como landing corporativa profesional.

Incluye:

- Backend en Laravel (API REST)
- Frontend en Vue 3 + TypeScript + Vite
- Formulario de contacto conectado a `/api/contact`
- Arquitectura SPA modular

---

## Tecnologías

### Backend
- Laravel
- API REST
- MySQL
- Validaciones y controladores

### Frontend
- Vue 3
- TypeScript
- Vite
- Axios

---

## Estructura del Proyecto

```
LynxOS-Consulting/
│
├── app/                # Backend Laravel
├── routes/
├── database/
├── frontend/           # SPA Vue
│   └── src/
│       ├── components/
│       ├── App.vue
│       └── main.ts
└── README.md
```

---

## Instalación y ejecución

### Backend

```
composer install
php artisan serve
```

### Frontend

```
cd frontend
npm install
npm run dev
```

## Funcionalidades

- Navbar con navegación SPA
- Hero corporativo
- Sección de servicios B2B
