# 🚀 Business Management Web Application (Laravel + Vue)

## 🌐 Demo online
https://polite-buttercream-6dca2d.netlify.app

Aplicación web full-stack basada en arquitectura API REST, desarrollada con Laravel (backend) y Vue (frontend).

El sistema permite gestionar entidades empresariales (clientes, productos, solicitudes, etc.) mediante un backend estructurado y un frontend desacoplado que consume los datos dinámicamente.

Incluye una versión compilada del frontend para despliegue estático (GitHub Pages) con fines de demostración.

---

## 🧱 Stack tecnológico

### Backend
- PHP
- Laravel
- Eloquent ORM
- MySQL
- API REST

### Frontend
- Vue.js
- Vite
- JavaScript
- Axios

---

## ⚙️ Funcionalidades

- API REST para gestión de datos empresariales
- CRUD completo (clientes, productos, solicitudes, etc.)
- Consumo de datos desde frontend
- Separación clara entre backend y frontend
- Arquitectura orientada a entorno real
- Versión demo estática
